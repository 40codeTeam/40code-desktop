# 40code桌面版
## 安装
```
git clone https://github.com/40codeTeam/40code-desktop.git
cd 40code-desktop
yarn config set registry https://registry.npm.taobao.org/
yarn config set ELECTRON_MIRROR http://npm.taobao.org/mirrors/electron/
yarn install
```
## 运行
```
yarn start
```
## 部署
### windows
```
yarn build
```
### mac
```
yarn buildmac
```
